# AfterMint Moono
사용자가 소유한 무노(LG UPlus 발행 NFT명) 컨트랙트 NFT 갤러리 및 간단한 게이밍 기능을 추가한 홍보용 애플리케이션.

## 주요 기능
- Kaikas 지갑 연동
- 사용자가 소유한 무노 NFT 갤러리
- NFT를 이용한 간단한 터치 게임 

## 개발 기간
- 2023.03.08 ~ 2023.05.17 (약 7주)

## 개발 인원
- iOS Developer : 1명

## 사용 기술 스택
- UIKit(Code base)
- MVVM
- Kaikas(web3 wallet)
- OpenSea API(NFT metadata)
- Nuke (이미지 캐싱)
- DifferenceKit
- SpriteKit
- Firebase Firestore
